# Projects Directory

## Directory Structure

Each project will have its own subdirectory with the following structure:

```
project-name/
├── README.md          # Project documentation
├── src/              # Source code files
├── docs/             # Additional documentation
├── tests/            # Test files
├── assets/           # Images, CSS, JavaScript files
├── database/         # SQL scripts and database files
└── .gitignore        # Git ignore file
```
